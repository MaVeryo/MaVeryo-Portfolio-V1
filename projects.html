Database Operations

In this Python project, my team and I used Jupyter Notebook and Pandas to analyze databases, focusing on Crypto Currencies. We cleaned and organized the data, examined distribution, trends, and correlation, and calculated various values like P-Value and F-Statistic. The project helped me learn teamwork, professional speaking, pitching ideas, and using Pandas for data analysis.

Web Search Bot

I created a Java "shop bot" for a class project to simplify online shopping. Using a 3-letter prefix and the item name, it quickly directed me to top shops for the desired item. Although the visual display was not fully implemented, testing with students showed a 10% increase in speed for finding items. Unfortunately, the code was later deleted as it was associated with my former high school account on a website.

Adventure Game

In my initial Python-based computer science project, I created a text-based video game featuring encounters with monsters in eight rooms. The game included a Monster class, a Player class, and strategic elements like navigating hallways and entering rooms. This experience ignited my programming passion, driven by the game's surprises and strategic upgrades, while also offering potential for expanded features like graphics and diverse player classes.
